Setup guide

Follow these steps once to connect the app to your Google Sheet. You only do this once.

Step 1 — Create a Google Sheet
Go to sheets.google.com → New spreadsheet. Name it "Maintenance Tracker".
Copy the ID from the URL: docs.google.com/spreadsheets/d/[THIS PART]/edit
Paste it into CONFIG.SHEET_ID in this file.
Step 2 — Enable the Sheets API
Go to console.cloud.google.com → New project → APIs & Services → Enable APIs.
Search for "Google Sheets API" and enable it.
Also enable "Google Identity Services".
Step 3 — Create OAuth credentials
APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID.
Application type: Web application.
Authorized JavaScript origins: add your hosting URL (e.g. https://yourteam.github.io).
Copy the Client ID → paste into CONFIG.CLIENT_ID.
Step 4 — Deploy the Render API server
Push the render-server/ folder to a GitHub repo.
Go to render.com → New Web Service → connect your repo.
Add environment variable: ANTHROPIC_API_KEY = your key from console.anthropic.com.
After deploy, copy your Render URL (e.g. https://maintenance-tracker-api.onrender.com).
Paste it into CONFIG.RENDER_URL in this file.
Step 5 — Host the file
Upload tracker.html to GitHub Pages, Netlify, or your own server.
Share the URL with ATC staff — that's it.
